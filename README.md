
Description
=====================

Config files with simple scripts that unpack/install/config all that stuff:
Vim, file managers, shells and many others.


Running
=====================

To run script:

1) Make script executable

	chmod +x debian-configurator.sh
2) Run it

	sudo ./debian-configurator.sh
